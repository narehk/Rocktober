## One-Liner

Execution copy of Change Order W-12144. Replace Cloudflare Worker with GitHub Actions repository_dispatch + Spotify oEmbed.

## Problem Statement

The Cloudflare Worker is required for votes, submissions, and song search. The user doesn't have a Cloudflare account, blocking all testing of the core gameplay loop.

## Proposed Solution

1. Vote + submission → GitHub Actions via `repository_dispatch` (browser triggers workflow, script validates + commits)
2. Song search → Spotify oEmbed (public, credential-free) with URL paste
3. Fine-grained GitHub PAT (contents:write, single repo) for browser dispatch
4. Concurrency groups prevent race conditions

## Acceptance Criteria

- [ ] Votes persist to round JSON via GitHub Actions workflow

- [ ] Song search works via Spotify URL paste + oEmbed
